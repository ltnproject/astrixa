❓ FAQ

Common questions.
Honest answers.

What is Astrixa?
+
Astrixa is a premium AI platform that allows you to use your own API keys to chat with any AI model from any provider in a single, unified interface.
Who is Astra?
+
Astra is the default AI personality of the Astrixa platform. While you can use any model, Astra provides the core experience and assistant features.
Is my data safe?
+
Yes. All your chats, settings, and API keys are stored locally in your browser (Astrixa Vault). We never see your data.
